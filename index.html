Welcome to something else. Welcome to xSWZ.

xSilverwareZ are an exciting new age art-concept project coin built on crypto-public-ledger blockchain technology. Built from simple, old-school, Tried&TrueSM SHA256D hybrid Proof of Stake code based on Peercoin and Bitcoin. Completely open source, completely free, both as in beer and as in speech.
